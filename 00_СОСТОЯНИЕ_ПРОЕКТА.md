## 5. Окружение и грабли (важно при продолжении)

- ОС: Windows. В среде **НЕТ** node/python/zip/libreoffice (python3 — заглушка Store). Сборка Excel/Word — через **COM-автоматизацию из PowerShell**.
- **Excel COM:** `Value2` принимает только строки → числа писать строками, затем `TextToColumns`; ширину столбцов задавать ДО объединения ячеек; PowerShell-функции вызывать без скобок/запятых (ловушка single-array-arg).
- **Запуск PS-скриптов с кириллицей:** `Invoke-Expression (Get-Content -Raw -Encoding UTF8 <путь>)`.
- **Проверка посторонних символов (CJK):** grep `-P` в этой среде падает на больших кодпоинтах — использовать .NET regex в PowerShell: `[regex]::Matches($t,'[　-鿿぀-ヿ]')`.
- **Запись файлов из PS без BOM:** `[System.IO.File]::WriteAllText($p,$txt,(New-Object System.Text.UTF8Encoding($false)))`.
- OneDrive: доступ к облачным исходникам требует запущенного OneDrive.exe.
