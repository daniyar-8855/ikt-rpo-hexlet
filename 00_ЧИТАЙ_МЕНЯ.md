# Проект: Рабочие учебные программы (РУП) по модулю ИКТ

**Уровень:** Колледж / ТиПО
**Особенность:** несколько специальностей/квалификаций — у каждой **свои часы, свой тематический план и своя РУП**.
**Цель:** РУП по модулю ИКТ для каждой специальности → курс в Moodle.

---

## Главный принцип

- **Содержание тем ИКТ — общее** (хранится один раз в `_ОБЩЕЕ/04_Банк_тем_ИКТ`).
- **Часы и распределение — индивидуальны** для каждой специальности.
- Под каждую специальность создаётся своя папка (копия `_ШАБЛОН_специальности`).

---

## Структура

```
ВСе для ИКТ/
├── _ОБЩЕЕ/                         ← общее для всех специальностей
│   ├── 01_Нормативные_общие/
│   │   ├── ГОСО/                   ← гос. стандарт
│   │   └── Прочие_нормы/           ← приказы, общие документы
│   ├── 02_Образцы_оформления/      ← ОБРАЗЕЦ РУП (структура и оформление)
│   ├── 03_Исходные_материалы/      ← учебники, пособия, источники
│   └── 04_Банк_тем_ИКТ/           ← МАСТЕР-банк тем (наполнение модуля)
│
└── Специальности/
    ├── _ШАБЛОН_специальности/      ← шаблон, копируем под каждую специальность
    │   ├── 01_Нормативные/         ← типовая программа, рабочий уч. план (часы!)
    │   ├── 02_Тематический_план/
    │   ├── 03_Рабочая_программа/   (Черновики / Готовое)
    │   └── 04_Курс_Moodle/         (Лекции/Практические/Тесты/Ресурсы)
    │
    └── <НазваниеСпециальности>/    ← создаётся под каждую специальность
```

---

## Куда что класть

| Что загружаете | Куда |
|---|---|
| ГОСО | `_ОБЩЕЕ/01_Нормативные_общие/ГОСО` |
| Образец РУП | `_ОБЩЕЕ/02_Образцы_оформления` |
| Учебники, методички | `_ОБЩЕЕ/03_Исходные_материалы` |
| Типовая программа конкретной специальности | `Специальности/<спец>/01_Нормативные` |
| Рабочий учебный план (где указаны часы) | `Специальности/<спец>/01_Нормативные` |

---

## Порядок работы

1. **Образец** → `_ОБЩЕЕ/02_Образцы_оформления`. Разбираю структуру.
2. **ГОСО + банк тем** → формирую общий перечень тем модуля ИКТ.
3. **По каждой специальности:** беру часы из её уч. плана → строю тематический план → собираю РУП по образцу.
4. **Курс Moodle** → по утверждённым темам каждой специальности.

> Файлы можно перетаскивать в чат или класть в папки.
> Список специальностей и их часы пришлите — заведу под каждую отдельную папку.
