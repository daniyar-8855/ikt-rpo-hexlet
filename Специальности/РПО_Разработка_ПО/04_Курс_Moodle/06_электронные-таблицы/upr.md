# Упражнения. Занятие 06. Электронные таблицы

## Упражнение 1. Относительная или абсолютная
В ячейке C2 формула `=B2*F1` (F1 — общий коэффициент). При копировании вниз результат «ломается». Как исправить?

**Критерий приёмки:** зафиксировать F1 → `=B2*$F$1`; объяснено, что относительная ссылка съезжает.

## Упражнение 2. Подбери функцию
Какой функцией посчитать: (а) сумму продаж, (б) средний балл, (в) число студентов с баллом ≥60, (г) «зачёт/незачёт» по баллу?

**Критерий приёмки:** СУММ; СРЗНАЧ; СЧЁТЕСЛИ(...;">=60"); ЕСЛИ(...).

## Упражнение 3. Найди ошибку в таблице данных
В таблице есть пустая строка посередине и объединённая ячейка в заголовке. Почему фильтр и сортировка работают неверно?

**Критерий приёмки:** пустая строка разрывает диапазон; объединённые ячейки ломают фильтр; нужна сплошная таблица.

## Упражнение 4. Составь формулу
Дана таблица: A — товар, B — цена, C — количество. Запиши формулу стоимости в D2 и итоговой суммы по всем товарам.

**Критерий приёмки:** `=B2*C2`; `=СУММ(D2:D...)`.

## Упражнение 5. Спланируй мини-дашборд
Какие 3–4 показателя и какой график ты выведешь для таблицы успеваемости группы? Обоснуй.

**Критерий приёмки:** показатели (средний балл, % сдавших, макс/мин) + уместный график (столбчатый/круговой) с обоснованием.

## СРС (самостоятельная работа, 2 ч)

**Тема:** Создание расчётной таблицы / мини-дашборда

Создай расчётную таблицу с формулами (сумма, среднее) и простой диаграммой. Используй обезличенные/тестовые данные; если применял ИИ — укажи, для чего именно.

**Критерий приёмки:** используются не менее 3 формул, есть диаграмма, данные обезличены. Работа сдаётся преподавателю (файл/отчёт) и засчитывается как СРС по теме.
