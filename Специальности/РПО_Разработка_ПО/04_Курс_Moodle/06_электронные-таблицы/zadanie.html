Задание к занятию 06. Расчёты и мини-дашборд в таблице

Вариант — по номеру в журнале (свой набор данных на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Создай таблицу с данными своего варианта (минимум 8 строк, заголовки колонок).
  2. Добавь расчётную колонку с формулой (используй относительную и абсолютную ссылку).
  3. Посчитай 4 показателя функциями: сумма, среднее, мин/макс, количество по условию (СЧЁТЕСЛИ).
  4. Добавь колонку с ЕСЛИ (например, статус по порогу) и построй график.
Продвинутая часть (+15 баллов): сделай мини-дашборд (3–4 показателя + график + условное форматирование) и объясни устно, что он показывает и какие выводы из него следуют.

Наборы данных по вариантам

  • В1. Успеваемость группы (студент, балл).
  • В2. Продажи интернет-магазина (товар, цена, кол-во).
  • В3. Бюджет учебного проекта (статья, план, факт).
  • В4. Посещаемость (студент, занятий посещено из 16).
  • В5. Метрики сайта (страница, визиты, отказы).
  • В6. Время на задачи (задача, оценка, факт, часы).

Как сдать: загрузи файл таблицы (.xlsx или ссылку на Google Sheets с доступом), укажи номер варианта. На занятии — короткая устная защита формул.

СРС (самостоятельная работа, 2 ч)

Тема СРС: Создание расчётной таблицы / мини-дашборда.
Задание: Создай расчётную таблицу с формулами (сумма, среднее) и простой диаграммой. Используй обезличенные данные; укажи, если применял ИИ.
Критерий приёмки: используются не менее 3 формул, есть диаграмма, данные обезличены.
