# Электронные таблицы для расчётов и данных

Таблицы — это не только бухгалтерия. Для разработчика это быстрый инструмент: посчитать метрики, проверить гипотезу на данных, построить график, прикинуть бюджет проекта — без написания кода. Понимание формул и работы с данными в Excel/Google Sheets — первый шаг к навыку работы с данными (data literacy), который нужен в любой IT-роли.

## Чему ты научишься

- вводить формулы и ссылаться на ячейки (относительные и абсолютные ссылки);
- применять базовые функции для расчётов и анализа;
- строить простой график и наводить порядок в таблице данных.

## Ячейки, ссылки, формулы

Любая формула начинается с `=`. Ссылка на ячейку — её адрес (`A1`, `B2`).

- **Относительная** ссылка (`A1`) сдвигается при копировании формулы.
- **Абсолютная** (`$A$1`) — фиксируется (например, ссылка на курс валюты).

```
=B2*C2        цена × количество
=B2*$F$1      цена × фиксированный коэффициент
```

## Базовые функции

| Функция | Что делает |
|---|---|
| `СУММ` / `SUM` | сумма диапазона |
| `СРЗНАЧ` / `AVERAGE` | среднее |
| `МИН`/`МАКС` | минимум/максимум |
| `СЧЁТ` / `COUNT` | количество чисел |
| `ЕСЛИ` / `IF` | условие (если…то…иначе) |

Пример: `=ЕСЛИ(D2>=60; "зачёт"; "незачёт")`.

## Данные и порядок

Таблица данных = «как база»: первая строка — заголовки колонок, дальше строки-записи, без пустых строк внутри. Тогда работают сортировка, фильтр и сводные таблицы.

### Мини-кейс
Нужно из списка из 200 студентов с баллами посчитать средний балл и число сдавших (≥60). Решение: `=СРЗНАЧ(B2:B201)` и `=СЧЁТЕСЛИ(B2:B201; ">=60")`. Следующий шаг: построить столбчатую диаграмму по результатам.

## Частые ошибки

**Ошибка:** при копировании формулы «съезжает» ссылка на общий коэффициент.
**Почему:** использована относительная ссылка вместо абсолютной.
**Как правильно:** фиксировать через `$` (`$F$1`).

**Ошибка:** пустые строки и объединённые ячейки внутри таблицы данных.
**Почему:** ломают сортировку, фильтр и формулы диапазонов.
**Как правильно:** сплошная таблица: заголовки + записи, без «дыр».

## Итог: что теперь делать

- Начинай формулу с `=`, понимай разницу относительной и абсолютной ссылки.
- Освой `СУММ/СРЗНАЧ/ЕСЛИ/СЧЁТЕСЛИ` — это 80% задач.
- Держи данные «как базу»: заголовки + записи, без пустых строк.
- Для наглядности строй график прямо из таблицы.
