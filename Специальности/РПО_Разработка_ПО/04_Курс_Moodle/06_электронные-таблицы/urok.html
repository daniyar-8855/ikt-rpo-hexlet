Электронные таблицы для расчётов и данных

Таблицы — это не только бухгалтерия. Для разработчика это быстрый инструмент: посчитать метрики, проверить гипотезу на данных, построить график — без написания кода. Это первый шаг к навыку работы с данными (data literacy), нужному в любой IT-роли.

Чему ты научишься: вводить формулы и ссылки (относительные/абсолютные); применять базовые функции; строить простой график и держать данные в порядке.

Ячейки, ссылки, формулы

Формула начинается с =. Ссылка — адрес ячейки (A1).

  • Относительная (A1) сдвигается при копировании.
  • Абсолютная ($A$1) фиксируется (например, курс валюты).
=B2*C2       цена × количество
=B2*$F$1     цена × фиксированный коэффициент

Базовые функции

ФункцияЧто делает
СУММ / SUMсумма диапазона
СРЗНАЧ / AVERAGEсреднее
МИН / МАКСминимум / максимум
СЧЁТЕСЛИ / COUNTIFколичество по условию
ЕСЛИ / IFусловие (если…то…иначе)

Пример: =ЕСЛИ(D2>=60; "зачёт"; "незачёт").

Данные и порядок

Таблица данных «как база»: первая строка — заголовки, дальше записи, без пустых строк внутри. Тогда работают сортировка, фильтр, сводные таблицы.

Мини-кейс

Из 200 студентов с баллами посчитать средний балл и число сдавших (≥60): =СРЗНАЧ(B2:B201) и =СЧЁТЕСЛИ(B2:B201; ">=60"). Дальше — столбчатая диаграмма.

Ошибка: при копировании формулы «съезжает» ссылка на общий коэффициент.
Почему: относительная ссылка вместо абсолютной.
Как правильно: фиксировать через $ ($F$1).
Ошибка: пустые строки и объединённые ячейки внутри таблицы данных.
Почему: ломают сортировку, фильтр и формулы.
Как правильно: сплошная таблица: заголовки + записи, без «дыр».
Итог — что теперь делать:
  • начинай формулу с =, различай относительную и абсолютную ссылку;
  • освой СУММ/СРЗНАЧ/ЕСЛИ/СЧЁТЕСЛИ — это 80% задач;
  • держи данные «как базу»: заголовки + записи, без пустых строк;
  • строй график прямо из таблицы.
