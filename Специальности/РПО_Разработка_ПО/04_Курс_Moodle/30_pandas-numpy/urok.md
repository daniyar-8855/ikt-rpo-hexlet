# Библиотеки Pandas и NumPy

Считать данные «вручную» циклами хорошо для понимания, но в реальности данных тысячи строк. Тогда подключают готовые библиотеки. **NumPy** быстро считает над массивами чисел, **Pandas** работает с таблицами (как Excel, но в коде). С ними одна строка заменяет целый цикл, а анализ данных становится по-настоящему быстрым.

## Чему ты научишься

- понимать, зачем нужны NumPy и Pandas;
- представлять таблицу как DataFrame;
- считать показатели по столбцу одной командой;
- фильтровать строки по условию.

## NumPy: массивы чисел

**NumPy** — библиотека для работы с массивами чисел. Массив похож на список, но операции над ним считаются быстро и сразу над всеми элементами:

```python
import numpy as np
ceny = np.array([1500, 2000, 1800])
print(ceny.mean())   # 1766.6 — среднее
print(ceny.sum())    # 5300
print(ceny * 2)      # [3000 4000 3600] — умножили все сразу
```

Не нужен цикл — `mean()`, `sum()`, умножение применяются ко всему массиву.

## Pandas: таблицы данных

**Pandas** работает с таблицами. Главный объект — **DataFrame** (таблица со столбцами и строками):

```python
import pandas as pd
data = {"tovar": ["Мышь", "Клав.", "Монитор"],
        "cena":  [1500, 2000, 9000]}
df = pd.DataFrame(data)
print(df["cena"].mean())   # среднее по столбцу
print(df["cena"].max())    # максимум
```

Столбец `df["cena"]` ведёт себя как список чисел, но с готовыми методами: `mean()`, `sum()`, `max()`, `min()`, `count()`.

## Фильтрация строк

Чтобы оставить строки по условию:

```python
dorogie = df[df["cena"] >= 2000]
print(dorogie)
```

Одна строка вместо цикла с `if` и `append`. Это и есть сила Pandas: меньше кода, понятнее результат.

### Мини-кейс
В таблице 1000 заказов со столбцом `summa`. Нужно: средний чек и число заказов дороже 5000. С Pandas это `df["summa"].mean()` и `len(df[df["summa"] > 5000])` — две строки вместо длинного цикла.

## Частые ошибки

**Ошибка:** забыть `import pandas as pd` / `import numpy as np`.
**Почему:** без импорта Python не знает этих имён — ошибка `NameError`.
**Как правильно:** импорт в начале файла.

**Ошибка:** обращаться к столбцу с опечаткой `df["Cena"]` вместо `df["cena"]`.
**Почему:** имена столбцов чувствительны к регистру — `KeyError`.
**Как правильно:** проверить точные имена через `df.columns`.

## Итог: что теперь делать

- Бери NumPy для быстрых расчётов над массивами чисел.
- Бери Pandas для таблиц: DataFrame = таблица со столбцами.
- Считай показатели по столбцу: `mean()`, `sum()`, `max()`.
- Фильтруй строки условием `df[df["поле"] > значение]`.
