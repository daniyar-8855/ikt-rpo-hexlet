Задание к занятию 30. Анализ таблицы с Pandas

Вариант — по номеру в журнале (своя таблица на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Создай DataFrame своего варианта (минимум 6 строк, 2–3 столбца).
  2. Посчитай по числовому столбцу: среднее, сумму, максимум, минимум.
  3. Отфильтруй строки по условию варианта одной командой.
  4. Выведи сколько строк осталось после фильтра (len).
  5. Кратко поясни код.
Продвинутая часть (+15 баллов): добавь новый вычисляемый столбец (например, цена со скидкой), отсортируй таблицу по нему и выведи топ-3. Защити устно.

Таблицы и условие фильтра по вариантам

  • В1. Товары (tovar, cena) — дороже среднего.
  • В2. Студенты (imya, ball) — балл ≥ 4.
  • В3. Заказы (nomer, summa) — сумма > 5000.
  • В4. Сотрудники (imya, staj) — стаж ≥ 3 лет.
  • В5. Города (gorod, naselenie) — население > 500000.
  • В6. Задачи (zadacha, chasy) — дольше 8 часов.
