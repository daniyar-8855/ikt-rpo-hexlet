Задание к занятию 14. Получение госуслуги через eGov / mGov

Вариант — по номеру в журнале (своя услуга на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Найди услугу своего варианта на eGov.kz и составь пошаговую инструкцию её получения (вход → поиск → заявление → подпись → результат).
  2. Приложи скриншоты ключевых шагов (можно на тестовом/демо-примере без передачи реальных данных).
  3. Укажи, какие данные требуются и какой способ подписи/входа.
  4. Опиши, как проверить правильность полученного документа.
Продвинутая часть (+15 баллов): запиши короткую видеоинструкцию (1–2 мин) получения услуги и защити устно. Не показывай реальные персональные данные.

Услуги по вариантам

  • В1. Адресная справка.
  • В2. Справка о несудимости.
  • В3. Регистрация ИП.
  • В4. Справка о регистрации по месту жительства членов семьи.
  • В5. Запись в очередь / получение талона.
  • В6. Справка о пенсионных отчислениях.

Как сдать: загрузи инструкцию со скриншотами (без реальных перс. данных), укажи номер варианта. На занятии — короткая устная защита.

СРС (самостоятельная работа, 2 ч)

Тема СРС: Видеоинструкция получения госуслуги через eGov.kz.
Задание: Опиши пошагово (или сними видео) получение одной госуслуги через eGov.kz. Используй обезличенные данные; укажи, если применял ИИ.
Критерий приёмки: показаны шаги вход→услуга→заявка→результат, использованы тестовые данные.
