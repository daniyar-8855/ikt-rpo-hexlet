# Упражнения. Занятие 03. Командная строка и терминал

> Формат — выполнение команд и разбор ошибок. Критерии приёмки под каждым заданием.

## Упражнение 1. Разбери команду на части
Дана команда: `cp -r ./src ./backup`. Назови: где команда, где флаг, где аргументы. Что она делает и зачем флаг `-r`?

**Критерий приёмки:** команда — `cp`, флаг — `-r` (рекурсивно, с подпапками), аргументы — `./src` и `./backup`; копирует папку src в backup.

## Упражнение 2. Собери структуру одной серией команд
Запиши команды, которые из текущей папки создадут:
```
project/
  src/
  tests/
  README.md
```
**Критерий приёмки:** последовательность вида `mkdir project` → `cd project` → `mkdir src tests` → `touch README.md` (или PowerShell-аналоги).

## Упражнение 3. Найди опасную/ошибочную команду
Что не так в каждой строке?
1. `rm -rf /`
2. `cd Мои документы`
3. `mkdir`

**Критерий приёмки:** 1 — удаление всей системы (катастрофа); 2 — пробел без кавычек; 3 — нет имени папки (аргумента).

## Упражнение 4. Переведи действие в команду
Запиши команду (bash ИЛИ PowerShell) для каждого действия:
1. узнать текущую папку;
2. показать все файлы, включая скрытые;
3. удалить файл `old.log`;
4. получить справку по команде создания папки.

**Критерий приёмки:** верные команды для выбранной оболочки (`pwd`; `ls -a`/`Get-ChildItem -Force`; `rm old.log`/`Remove-Item old.log`; `mkdir --help`/`Get-Help mkdir`).

## Упражнение 5. Восстанови порядок действий
Студент хочет безопасно удалить файлы в папке проекта. Расставь шаги в правильном порядке: `rm`, `ls`, `pwd`. Объясни, почему именно так.

**Критерий приёмки:** порядок `pwd` → `ls` → `rm`; обоснование: сначала убедиться в местоположении и содержимом, потом удалять.

## СРС (самостоятельная работа, 2 ч)

**Тема:** Практикум: базовые команды терминала (отчёт)

Выполни 8–10 базовых команд терминала и оформи отчёт со скриншотами. Используй обезличенные/тестовые данные; если применял ИИ — укажи, для чего именно.

**Критерий приёмки:** показаны команды навигации и работы с файлами, есть скриншоты и пояснения. Работа сдаётся преподавателю (файл/отчёт) и засчитывается как СРС по теме.
