Задание к занятию 03. Серия команд для задачи

Вариант — по номеру в журнале (одна задача на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Выполни задачу своего варианта только командами терминала (bash или PowerShell — на выбор).
  2. Запиши все команды по порядку с коротким комментарием к каждой.
  3. Приложи скриншоты или скринкаст выполнения (видно результат каждой команды).
  4. Отметь, где использовал флаги и зачем.
Продвинутая часть (+15 баллов): собери свои команды в один скрипт (`.sh` или `.ps1`) и запусти его одним вызовом. Объясни устно, что делает скрипт.

Задачи по вариантам

  • В1. Создать структуру `web/` → `src`, `assets`, `tests` + файл `index.html`; вывести дерево.
  • В2. Создать `bot/` → `src`, `config`, `logs` + файл `bot.py`; скопировать `bot.py` в `backup/`.
  • В3. Создать `api/` → `src`, `db`, `tests` + `README.md`; переименовать `db` в `migrations`.
  • В4. Создать `game/` → `src`, `assets/sprites`, `levels` + `main.js`; удалить пустую папку `levels`.
  • В5. Создать `data/` → `raw`, `clean`, `reports` + `notebook.ipynb`; вывести только скрытые файлы папки.
  • В6. Создать `app/` → `src`, `docs`, `tests`; скопировать всю `src` в `src_backup` рекурсивно.

Как сдать: загрузи отчёт с командами и скриншотами, укажи номер варианта. На занятии — короткая устная защита (выполнить 1–2 команды по просьбе преподавателя).

СРС (самостоятельная работа, 2 ч)

Тема СРС: Практикум: базовые команды терминала (отчёт).
Задание: Выполни 8–10 базовых команд терминала и оформи отчёт со скриншотами. Используй обезличенные данные; укажи, если применял ИИ.
Критерий приёмки: показаны команды навигации и работы с файлами, есть скриншоты и пояснения.
