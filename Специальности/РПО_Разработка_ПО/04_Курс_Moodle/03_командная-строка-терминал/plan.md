# План занятия 03. Командная строка и терминал

> Для преподавателя (в Moodle — скрытая «Страница»). Тип: лабораторно-практическое, 2 ч. РО 2.1.

- **Дисциплина:** Применение ИКТ и цифровых технологий (РПО, 4S06130103)
- **Тема:** Командная строка и терминал
- **Цель:** научить уверенно работать в терминале — навигация, управление файлами, чтение команд.
- **Результат обучения:** РО 2.1.

## Ожидаемые результаты
Студент умеет: объяснить роль терминала; перемещаться по файловой системе; создавать/копировать/удалять файлы командами; читать структуру команды и находить справку.

## Ход занятия (90 мин)
| Этап | Время | Содержание |
|---|---|---|
| Орг. момент | 5 мин | Тема, цель, зацепка «как развернуть проект на сервере без мышки» |
| Теория | 20 мин | Зачем терминал; структура команды; bash vs PowerShell |
| Демонстрация | 20 мин | Базовый набор команд вживую, создание структуры проекта |
| Практика | 30 мин | Упражнения 2–5 (сборка структуры, разбор ошибок, перевод действий в команды) |
| Выдача задания | 10 мин | Серия команд по вариантам + скриншоты |
| Итог, рефлексия | 5 мин | Что делать / где применяется / как не ошибиться |

## Средства обучения
Проектор, ПК с терминалом (PowerShell и/или WSL/Linux), демонстрационный экран.

## Контроль
- Текущий: упражнения на занятии, тест (банк 12 вопросов, 10 случайных).
- Задание: серия команд по вариантам + устная защита.

## Источники
- ГОСО ТиПО (приказ МП РК № 348); Ubuntu CLI tutorial; Microsoft Learn (PowerShell).

## Адаптация (при необходимости)

При наличии обучающихся с особыми образовательными потребностями предусматриваются адаптация темпа, дополнительные пояснения и индивидуальные задания, одобренные методическим объединением.
