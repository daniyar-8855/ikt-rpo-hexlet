Задание к занятию 31. Визуализация набора данных

Вариант — по номеру в журнале (свой набор данных на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Возьми набор данных своего варианта (6–8 значений).
  2. Выбери подходящий тип графика и обоснуй выбор.
  3. Построй график (Matplotlib или таблица-генератор/онлайн-инструмент).
  4. Добавь заголовок и подписи осей.
  5. Сделай вывод: что видно на графике (тренд, лидер, аномалия).
Продвинутая часть (+15 баллов): построй второй график другого типа по тем же данным и объясни, какой честнее и почему. Покажи пример «обманывающей» оси и исправь её. Защити устно.

Наборы данных и рекомендуемый акцент по вариантам

  • В1. Продажи по городам — сравнение категорий.
  • В2. Температура по дням недели — динамика.
  • В3. Структура расходов студента — доли.
  • В4. Посещаемость по месяцам — динамика.
  • В5. Оценки по предметам — сравнение категорий.
  • В6. Время на задачи в проекте — сравнение/доли.
