Задание к занятию 17. Цифровые сервисы для самозанятого разработчика

Вариант — по номеру в журнале (свой сценарий на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Составь пошаговую инструкцию онлайн-регистрации ИП по сценарию варианта (вход → заявление → подпись → результат).
  2. Опиши 3 функции налогового кабинета, которые понадобятся разработчику.
  3. Разбери рискованную ситуацию варианта: что не так и как сделать легально/безопасно.
  4. Кратко поясни, почему официальное оформление выгодно (3 аргумента).
Продвинутая часть (+15 баллов): найди актуальную информацию о льготных налоговых режимах для ИТ в РК и кратко изложи (со ссылкой на источник). Защищается устно.

Сценарии по вариантам

  • В1. Фрилансер берёт первый крупный заказ; риск: оплата на личную карту без договора.
  • В2. Разработчик делает свой продукт и хочет продавать; риск: работа без оформления.
  • В3. Студент подрабатывает версткой сайтов; риск: регистрация через посредника с передачей ЭЦП.
  • В4. Команда хочет выставлять счета клиентам; риск: нет закрывающих документов.
  • В5. Самозанятый забывает про отчётность; риск: задолженность и штрафы.
  • В6. Разработчик работает с зарубежным клиентом; риск: непонимание налогообложения.
