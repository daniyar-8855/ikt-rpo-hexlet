# Упражнения. Занятие 24. Интерактивные формы и совместная работа

> Формат — проектирование и выбор сервиса. Критерии приёмки под каждым заданием.

## Упражнение 1. Подбери поля формы
Для регистрации на хакатон какие поля нужны, а какие лишние: ФИО, ИИН, e-mail, размер обуви, команда, телефон?

**Критерий приёмки:** нужны — ФИО, e-mail, команда, телефон; лишние — ИИН, размер обуви.

## Упражнение 2. Тип поля
Подбери тип поля: (а) выбрать один вариант; (б) оценка по шкале 1–5; (в) свободный комментарий; (г) e-mail.

**Критерий приёмки:** (а) один из списка; (б) шкала; (в) текст; (г) поле e-mail с проверкой.

## Упражнение 3. Выбери сервис
Что использовать: (а) планировать задачи спринта; (б) собрать опрос; (в) совместно писать документ; (г) быстрая коммуникация команды?

**Критерий приёмки:** (а) доска задач/Trello; (б) Google Forms; (в) Google Docs; (г) чат/мессенджер.

## Упражнение 4. Наведи порядок в команде
Команда ведёт всё в личке. Предложи структуру: где задачи, где обсуждения, где документы.

**Критерий приёмки:** доска задач + общий чат + облако с документами; обоснование.

## Упражнение 5. Приватность формы
Какие правила соблюсти при сборе данных через форму (минимум 3)?

**Критерий приёмки:** минимизация полей, указание цели сбора, не запрашивать лишние перс. данные, ограниченный доступ к ответам.
