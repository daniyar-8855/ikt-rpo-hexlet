Задание к занятию 24. Форма + командное пространство

Вариант — по номеру в журнале (свой сценарий на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Создай онлайн-форму под сценарий своего варианта: подбери поля и их типы (минимизация данных).
  2. Приложи ссылку/скриншот формы и пример таблицы ответов.
  3. Опиши командное пространство для проекта: где задачи, где обсуждения, где документы (с конкретными сервисами).
  4. Укажи правила приватности формы (что не запрашиваем, кто видит ответы).
Продвинутая часть (+15 баллов): настрой реальную доску задач (Trello/таблица) для проекта с 3 колонками и парой карточек; приложи ссылку/скриншот, защити устно.

Сценарии по вариантам

  • В1. Регистрация на хакатон.
  • В2. Опрос обратной связи по курсу.
  • В3. Запись на консультацию к преподавателю.
  • В4. Сбор заявок в команду проекта.
  • В5. Тест/квиз по теме (с автопроверкой).
  • В6. Анкета пользователей для учебного продукта.
