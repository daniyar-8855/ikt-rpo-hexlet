Задание к занятию 18. Лицензирование для ИТ-продукта

Вариант — по номеру в журнале (свой продукт на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Определи для продукта своего варианта: требуется ли лицензия/разрешение и кому (разработчику или клиенту/сфере применения).
  2. Проверь на eLicense.kz, относится ли сфера к лицензируемым; укажи, что нашёл.
  3. Если требуется — опиши порядок получения онлайн (вход → выбор → заявление → подпись → результат).
  4. Разбери 1 риск (например, посредники) и как действовать правильно.
Продвинутая часть (+15 баллов): составь чек-лист «Нужна ли моему продукту лицензия?» (5–6 вопросов) и защити устно.

Продукты по вариантам

  • В1. Сайт-визитка для кафе.
  • В2. Приложение для аптеки с продажами.
  • В3. Платформа онлайн-курсов (образование).
  • В4. Финтех-приложение для переводов.
  • В5. Telegram-бот для записи в барбершоп.
  • В6. Сервис для частной клиники (запись + данные пациентов).

СРС (самостоятельная работа, 2 ч)

Тема СРС: Инструкция получения лицензии/разрешения.
Задание: Составь инструкцию получения лицензии или разрешения через eLicense.kz. Используй обезличенные данные; укажи, если применял ИИ.
Критерий приёмки: описаны шаги подача→проверка→выдача, указан тип документа.
