# План учебного занятия

**Организация:** ТОО «Колледж Хекслет Казахстан»  
**Дисциплина:** Применение информационно-коммуникационных и цифровых технологий  
**Тема занятия:** Разработать мини-проект с применением ИИ
**Подготовил педагог:** ____________________  
**Дата:** «___» __________ 20__ г.

## 1. Общие сведения

- **Курс, группа:** ____________________ (квалификация 4S06130103 «Разработчик ПО»)
- **Тип занятия:** урок применения знаний (практикум) / проектная работа
- **Продолжительность:** 2 часа (90 мин) — 1 пара
- **Результат обучения:** РО 2.3 · **Связь с ПМ:** Разработка программных модулей

## 2. Цели, задачи

**Цель:** Научить выполнять мини-проект с применением ИИ от постановки задачи до проверенного результата, показать связь с профессией (портфолио) и ответственное применение ИИ.

**Задачи:**
- раскрыть метод проекта и этапы мини-проекта (идея → данные → инструмент ИИ → результат → вывод и проверка);
- научить выбирать задачу и подбирать инструмент курса под её тип;
- сформировать навык проверки результата ИИ и обезличивания данных;
- показать, как оформить работу для портфолио и честно указать вклад ИИ.

## 3. Ожидаемые результаты

Обучающийся:
- называет этапы мини-проекта по порядку;
- выбирает инструмент курса под тип задачи и применяет ИИ;
- проверяет результат ИИ и использует тестовые/обезличенные данные;
- оформляет результат и указывает, где и как использован ИИ.

## 4. Необходимые ресурсы

- Курс в Moodle: страница «Теория», тест и итоговое проектное задание занятия 38.
- Схемы: «Этапы мини-проекта», «Чек-лист готовности» (папка assets).
- Компьютер, проектор, доступ к Moodle и интернету.

## 5. Ход занятия

| Этап / Время | Действия педагога | Действия обучающегося |
|---|---|---|
| Организационный момент (5 мин) | Приветствие, проверка готовности и присутствующих | Занимают места, готовятся к занятию |
| Мотивация (10 мин) | Вводит практической ситуацией (выгрузка заказов, готовый «продукт» для портфолио), задаёт вопросы | Обсуждают, предлагают варианты задач |
| Целеполагание (5 мин) | Объявляет тему, цели и формат проектной работы | Записывают тему, формулируют ожидания |
| Изучение нового материала (35 мин) | Разбирает этапы мини-проекта, демонстрирует схемы, таблицу инструментов и мини-кейс, акцент на проверке ИИ и обезличивании | Слушают, работают со схемами, выбирают тему проекта |
| Закрепление (15 мин) | Организует практикум: ученики планируют этапы своего мини-проекта, разбор типичной ошибки | Планируют проект, выполняют упражнения |
| Контроль (10 мин) | Запускает тест в Moodle | Проходят тест самопроверки |
| Итог и рефлексия (5 мин) | Возвращает к вопросам: этапы, проверка ИИ, ответственность | Отвечают, подводят итог |
| Задание (5 мин) | Поясняет итоговое проектное задание, варианты и критерии | Записывают вариант, уточняют |
