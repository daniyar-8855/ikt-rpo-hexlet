# Практическая работа (Задание Moodle). Тема 3. Командная строка / терминал

> Элемент Moodle: «Задание» (Assignment). Тип занятия: лабораторно-практическое, 2 ч. РО 2.1.
> Связь с ПМ: все профессиональные модули (рабочая среда разработчика).

## Цель
Освоить базовые команды терминала (Windows PowerShell / Linux bash) для навигации, работы с файлами и запуска программ.

## Теоретический минимум
Терминал — текстовый интерфейс управления ОС. Разработчик использует его ежедневно: git, запуск скриптов, установка пакетов, навигация по проекту.

## Задачи
1. Открыть терминал (PowerShell или bash) и определить текущую директорию (`pwd` / `Get-Location`).
2. Создать структуру папок проекта: `project/`, внутри — `src/`, `docs/`, `data/`.
3. Создать файл `README.txt` и записать в него название проекта.
4. Вывести список файлов с деталями (`ls -l` / `dir`).
5. Скопировать `README.txt` в папку `docs/` и переименовать в `about.txt`.
6. Показать дерево каталогов проекта.
7. Узнать справку по любой команде (`man` / `Get-Help` / `--help`).

## Что сдать (загрузить в Moodle)
- **Скринкаст (видео 2–4 мин)** или **серию скриншотов** с выполнением всех 7 задач.
- Текстовый файл с использованными командами и кратким комментарием к каждой.

## Критерии оценивания (рубрика)
| Критерий | Балл |
|---|---|
| Выполнены все 7 задач | 40 |
| Команды применены верно, без ошибок | 30 |
| Есть комментарии-пояснения к командам | 20 |
| Аккуратность оформления отчёта | 10 |
| **Итого** | **100** |
