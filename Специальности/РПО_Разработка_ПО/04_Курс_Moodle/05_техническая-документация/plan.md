# План занятия 05. Техническая документация в текстовом редакторе

> Для преподавателя (в Moodle — скрытая «Страница»). Тип: комбинированное, 2 ч. РО 2.1.

- **Дисциплина:** Применение ИКТ и цифровых технологий (РПО, 4S06130103)
- **Тема:** Техническая документация в текстовом редакторе
- **Цель:** научить структурно оформлять техническую документацию (README, инструкции).
- **Результат обучения:** РО 2.1.

## Ожидаемые результаты
Студент умеет: отличать техдокументацию от «текста ради текста»; применять стили заголовков, списки, таблицы, код-блоки; составить README по схеме.

## Ход занятия (90 мин)
| Этап | Время | Содержание |
|---|---|---|
| Орг. момент | 5 мин | Зацепка: «README из одной строки — проект не запустить» |
| Теория | 20 мин | Назначение документации; структура; элементы оформления |
| Демонстрация | 15 мин | Сборка README по схеме вживую |
| Практика | 35 мин | Упражнения 1–5 (оценка, переписывание, каркас README) |
| Выдача задания | 10 мин | README проекта по вариантам |
| Итог, рефлексия | 5 мин | Что делать / где применяется / как не ошибиться |

## Средства обучения
Проектор, ПК с текстовым редактором (Word/Google Docs) и редактором кода (Markdown).

## Контроль
- Текущий: упражнения, тест (банк 12, 10 случайных).
- Задание: README проекта по вариантам + устная защита.

## Источники
- ГОСО ТиПО (приказ МП РК № 348); GitHub Docs (Markdown); справка Google Документы.

## Адаптация (при необходимости)

При наличии обучающихся с особыми образовательными потребностями предусматриваются адаптация темпа, дополнительные пояснения и индивидуальные задания, одобренные методическим объединением.
