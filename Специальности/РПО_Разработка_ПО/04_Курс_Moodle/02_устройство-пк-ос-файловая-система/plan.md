# План занятия 02. Устройство ПК, ОС и файловая система

> Для преподавателя (в Moodle — скрытая «Страница»). Тип: лекция, 2 ч. РО 2.1.

- **Дисциплина:** Применение ИКТ и цифровых технологий (РПО, 4S06130103)
- **Тема:** Устройство ПК, операционные системы (Windows/Linux) и файловая система
- **Цель:** сформировать понимание аппаратной и системной среды разработчика и навык ориентации в файловой системе.
- **Результат обучения:** РО 2.1.

## Ожидаемые результаты
Студент умеет: подобрать ПК под задачи разработки; объяснить роль ОС и место Linux; различать абсолютные/относительные пути; организовать структуру проекта и избегать ошибок с именами файлов.

## Ход занятия (90 мин)
| Этап | Время | Содержание |
|---|---|---|
| Орг. момент | 5 мин | Тема, цель, зацепка «работает у меня — не работает у него» |
| Теория | 25 мин | Компоненты ПК; что важно разработчику (RAM/SSD) |
| Теория | 20 мин | ОС: функции; Windows vs Linux; WSL/контейнеры |
| Теория + разбор | 20 мин | Файловая система: пути, регистр, структура проекта; мини-кейс |
| Практика | 15 мин | Упражнения 2 и 3 (пути; поиск ошибок регистра/имён) |
| Выдача задания | 3 мин | Окружение и структура проекта (варианты по подгруппам) |
| Итог, рефлексия | 2 мин | Что делать / где применяется / как не ошибиться |

## Средства обучения
Проектор, ПК с Windows и доступом к Linux/WSL, демонстрация терминала и файлового менеджера.

## Контроль
- Текущий: упражнения на занятии, тест (банк 14 вопросов, выдаётся 10 случайных).
- Задание: проектирование окружения и структуры проекта (6 вариантов) + устная защита.

## Источники
- ГОСО ТиПО (приказ МП РК № 348).
- Документация Microsoft Learn (WSL), учебные материалы по Linux.

## Заметки для преподавателя
- ФИО преподавателя, дата — заполнить из расписания.
- Показать вживую разницу регистра в Linux (`File.js` vs `file.js`).
