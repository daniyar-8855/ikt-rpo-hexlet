Задание к занятию 19. Работа с открытыми данными

Вариант — по номеру в журнале (своя тема данных на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Найди на data.egov.kz набор данных по теме своего варианта; укажи название, формат, период, ссылку.
  2. Скачай и опиши структуру: какие колонки/поля, сколько записей.
  3. Сделай 3 вывода из данных (мини-анализ: максимум, минимум, динамика и т.п.).
  4. Объясни своими словами, как эти данные можно было бы получать через API.
Продвинутая часть (+15 баллов): построй простой график/дашборд по набору (в таблицах или коде) и защити выводы устно.

Темы данных по вариантам

  • В1. Население по регионам.
  • В2. Образование (школы/колледжи/студенты).
  • В3. Здравоохранение (учреждения/показатели).
  • В4. Экономика/занятость.
  • В5. Цифровизация/связь/интернет.
  • В6. Транспорт/ЖКХ.
