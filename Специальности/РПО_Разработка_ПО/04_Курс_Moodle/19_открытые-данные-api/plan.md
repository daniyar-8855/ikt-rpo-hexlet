**Организация:** ТОО «Колледж Хекслет Казахстан»  
**Дисциплина:** Применение информационно-коммуникационных и цифровых технологий  
**Тема занятия:** Использовать порталы открытых данных (data.egov.kz) и их API

**Дата:** «___» __________ 20__ г.

## 1. Общие сведения

- **Курс, группа:** ____________________ (квалификация 4S06130103 «Разработчик ПО»)
- **Тип занятия:** комбинированное (лабораторно-практическое)
- **Продолжительность:** 2 часа (90 мин)
- **Результат обучения:** РО 2.2 · **Связь с ПМ:** Работа с данными и интеграции

## 2. Цели, задачи

**Цель:** Научить находить и использовать открытые данные (data.egov.kz), сформировать понимание принципа работы API и форматов данных, показать связь с будущей профессией.

**Задачи:**
- научить находить и скачивать наборы открытых данных на data.egov.kz;
- раскрыть понятие API и схему обмена «запрос → ответ JSON»;
- научить различать форматы CSV, JSON, XLSX и оценивать их пригодность;
- сформировать понимание различия между открытыми и персональными данными.

## 3. Ожидаемые результаты

Обучающийся:
- находит и скачивает наборы открытых данных на data.egov.kz;
- объясняет, что такое API и как идёт обмен «запрос → ответ JSON»;
- различает форматы CSV, JSON, XLSX и выбирает подходящий;
- не путает открытые данные с персональными.

## 4. Необходимые ресурсы

- Курс в Moodle: страница «Теория», тест и задание занятия 19.
- Схемы: «Что такое открытые данные», «Обращение к открытым данным через API» (папка assets).
- Компьютер, проектор, доступ к Moodle, интернету и порталу data.egov.kz; табличный редактор.
