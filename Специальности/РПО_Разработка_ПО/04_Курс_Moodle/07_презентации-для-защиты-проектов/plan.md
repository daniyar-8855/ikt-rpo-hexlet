# План занятия 07. Презентации для защиты проектов

> Для преподавателя (в Moodle — скрытая «Страница»). Тип: комбинированное, 2 ч. РО 2.1.

- **Дисциплина:** Применение ИКТ и цифровых технологий (РПО, 4S06130103)
- **Тема:** Презентации для защиты проектов
- **Цель:** научить готовить презентацию и устную защиту проекта.
- **Результат обучения:** РО 2.1.

## Ожидаемые результаты
Студент умеет: строить презентацию по схеме «проблема → решение → результат»; оформлять слайды читаемо; готовить устное выступление и демонстрацию.

## Ход занятия (90 мин)
| Этап | Время | Содержание |
|---|---|---|
| Орг. момент | 5 мин | Зацепка: «хороший проект — слабая защита» |
| Теория | 20 мин | Роль слайдов; структура защиты; правила оформления |
| Разбор | 15 мин | Анализ «плохого» и «хорошего» слайда |
| Практика | 35 мин | Упражнения 1–5 (ошибки слайда, структура, тезисы, демо-план) |
| Выдача задания | 10 мин | Презентация защиты по вариантам |
| Итог, рефлексия | 5 мин | Что делать / где применяется / как не ошибиться |

## Средства обучения
Проектор, ПК с PowerPoint/Google Презентациями, примеры удачных/неудачных слайдов.

## Контроль
- Текущий: упражнения, тест (банк 12, 10 случайных).
- Задание: презентация + текст выступления по вариантам; живая защита (продвинутая часть).

## Источники
- ГОСО ТиПО (приказ МП РК № 348); справки PowerPoint / Google Презентации / Canva.

## Адаптация (при необходимости)

При наличии обучающихся с особыми образовательными потребностями предусматриваются адаптация темпа, дополнительные пояснения и индивидуальные задания, одобренные методическим объединением.
