# План учебного занятия

**Дисциплина:** Применение информационно-коммуникационных и цифровых технологий  
**Тема занятия:** Создавать презентации для защиты проектов
**Подготовил педагог:** ____________________  
**Дата:** «___» __________ 20__ г.

## 1. Общие сведения

- **Курс, группа:** ____________________ (квалификация 4S06130103 «Разработчик ПО»)
- **Тип занятия:** комбинированное
- **Продолжительность:** 2 часа (90 мин)
- **Результат обучения:** РО 2.1 · **Связь с ПМ:** Разработка программных модулей

## 2. Цели, задачи

**Цель:** Научить готовить презентацию и устную защиту проекта по логике «проблема → решение → результат», показать связь с будущей профессией.

**Задачи:**
- объяснить роль слайда как опоры для речи, а не её замены;
- раскрыть структуру защиты проекта (титул → проблема → решение → демонстрация → итог);
- сформировать три правила хорошего слайда (мало текста, один смысл, контраст);
- показать важность демонстрации работы продукта.

## 3. Ожидаемые результаты

Обучающийся:
- строит презентацию по схеме «проблема → решение → результат»;
- оформляет слайды читаемо: один смысл, мало текста, крупный шрифт;
- готовит устное выступление, а не зачитывание слайдов;
- показывает работу продукта через демо или скриншоты.

## 4. Необходимые ресурсы

- Курс в Moodle: страница «Теория», тест и задание занятия 07.
- Схемы: «Структура презентации», «Плохой и хороший слайд» (папка assets).
- Компьютер, проектор, ПК с PowerPoint/Google Презентациями, доступ к Moodle и интернету.

## 5. Ход занятия

| Этап / Время | Действия педагога | Действия обучающегося |
|---|---|---|
| Организационный момент (5 мин) | Приветствие, проверка готовности и присутствующих | Занимают места, готовятся к занятию |
| Мотивация (10 мин) | Вводит практической ситуацией («хороший проект — слабая защита»), задаёт вопросы | Обсуждают, предлагают варианты |
| Целеполагание (5 мин) | Объявляет тему и цели занятия | Записывают тему, формулируют ожидания |
| Изучение нового материала (35 мин) | Раскрывает роль слайда, структуру защиты и правила оформления, демонстрирует схемы, приводит мини-кейс | Слушают, работают со схемами, задают вопросы |
| Закрепление (15 мин) | Организует практику (упражнения, разбор «плохого» и «хорошего» слайда) и разбор типичной ошибки | Выполняют задания, отвечают |
| Контроль (10 мин) | Запускает тест в Moodle (банк 12, 10 случайных) | Проходят тест самопроверки |

| Задание (5 мин) | Поясняет вариантное задание и критерии | Записывают, уточняют |
