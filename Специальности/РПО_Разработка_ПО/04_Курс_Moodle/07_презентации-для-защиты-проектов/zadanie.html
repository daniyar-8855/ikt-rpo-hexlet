Задание к занятию 07. Презентация защиты проекта

Вариант — по номеру в журнале (свой проект для презентации на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Сделай презентацию 5–8 слайдов по схеме: титул → проблема → решение → технологии → результат → итог.
  2. На слайдах — тезисы и схемы, а не «простыни» текста; крупный шрифт, единый стиль.
  3. Добавь хотя бы одну схему (как работает продукт) и слайд с демонстрацией/скриншотами.
  4. Подготовь текст устного выступления на 3–4 минуты (отдельно от слайдов).
Продвинутая часть (+15 баллов): проведи живую защиту перед группой (3–4 мин) и ответь на 1–2 вопроса. Оценивается умение объяснять, а не читать слайды.

Темы проектов по вариантам

  • В1. Веб-сервис записи на консультации.
  • В2. Телеграм-бот напоминаний.
  • В3. Мобильное приложение «Расписание».
  • В4. Игра-викторина.
  • В5. Сервис анализа данных.
  • В6. Интернет-магазин (учебный).

Критерии оценивания (100 баллов)

КритерийБалл
Структура по схеме (5–8 слайдов)25

Текст устного выступления (3–4 мин)15

Итого100
Как сдать: загрузи файл презентации (.pptx или ссылку) и текст выступления, укажи номер варианта. Защита — устно на занятии.
