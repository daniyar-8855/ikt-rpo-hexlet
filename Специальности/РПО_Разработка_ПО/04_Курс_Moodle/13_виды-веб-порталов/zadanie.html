Задание к занятию 13. Карта веб-порталов под задачи

Вариант — по номеру в журнале (свой набор задач на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Для 5 задач своего варианта подбери подходящий портал и укажи его тип (информационный/справочный/интерактивный).
  2. Оформи как таблицу: задача → портал → тип → что оставляешь (данные/действие).
  3. Для каждого интерактивного портала перечисли признаки, по которым ты это определил.
  4. Отметь, где требуется ЭЦП/вход и где есть риск фишинга.
Продвинутая часть (+15 баллов): составь памятку «Как выбрать правильный портал и не попасть на двойник» (алгоритм из 5 шагов). Защищается устно.

Наборы задач по вариантам

  • В1. Узнать закон; получить адресную справку; оплатить штраф; найти статистику; прочитать новости.
  • В2. Зарегистрировать ИП; узнать НПА; скачать датасет; оплатить налог; найти вакансию.
  • В3. Получить справку о несудимости; узнать текст ГОСО; оплатить ЖКХ; найти учебник; прочитать гайд.
  • В4. Оформить лицензию; найти открытые данные; узнать закон; записаться к врачу; прочитать обзор.
  • В5. Оплатить госпошлину; узнать НПА; найти курс; получить выписку; прочитать инструкцию.
  • В6. Подать заявление онлайн; скачать датасет; узнать закон; перевести деньги; найти документацию.

Как сдать: загрузи отчёт-таблицу, укажи номер варианта. На занятии — короткая устная защита выбора.

СРС (самостоятельная работа, 2 ч)

Тема СРС: Классификация веб-порталов РК (таблица).
Задание: Составь таблицу-классификацию 8–10 веб-порталов РК по типам. Используй обезличенные данные; укажи, если применял ИИ.
Критерий приёмки: порталы распределены по типам, указано назначение каждого.
