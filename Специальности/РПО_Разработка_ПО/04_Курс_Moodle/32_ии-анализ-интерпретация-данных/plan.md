# План учебного занятия

**Дисциплина:** Применение информационно-коммуникационных и цифровых технологий  
**Тема занятия:** Использовать ИИ для анализа и интерпретации данных
**Подготовил педагог:** ____________________  
**Дата:** «___» __________ 20__ г.

## 1. Общие сведения

- **Курс, группа:** ____________________ (квалификация 4S06130103 «Разработчик ПО»)
- **Тип занятия:** комбинированное
- **Продолжительность:** 2 часа (90 мин)
- **Результат обучения:** РО 2.3 · **Связь с ПМ:** Работа с данными и БД

## 2. Цели, задачи

**Цель:** Научить применять ИИ для интерпретации данных, формулировать корректный запрос для анализа и критически проверять выводы ИИ при обязательном обезличивании персональных данных.

**Задачи:**
- показать, что ИИ умеет на этапе осмысления данных (показатели, тренды, гипотезы);
- научить строить запрос для анализа из трёх частей: данные + контекст + конкретный вопрос;
- сформировать привычку относиться к выводу ИИ как к гипотезе и проверять её по данным;
- закрепить правило обезличивания персональных данных перед отправкой в ИИ.

## 3. Ожидаемые результаты

Обучающийся:
- использует ИИ, чтобы объяснить показатели и найти закономерности;
- формулирует запрос для анализа из данных, контекста и конкретного вопроса;
- проверяет выводы ИИ по данным и фактам, не принимая их на веру;
- обезличивает данные перед отправкой в ИИ и объясняет, почему это требование закона.

## 4. Необходимые ресурсы

- Курс в Moodle: страница «Теория», тест и задание занятия 32.
- Схемы: «Структура запроса для анализа», «Вывод ИИ как гипотеза → проверка» (папка assets).
- Компьютер, проектор, доступ к Moodle и интернету.

## 5. Ход занятия

| Этап / Время | Действия педагога | Действия обучающегося |
|---|---|---|
| Организационный момент (5 мин) | Приветствие, проверка готовности и присутствующих | Занимают места, готовятся к занятию |
| Мотивация (10 мин) | Вводит практической ситуацией (таблица продаж: «что это значит и что делать?»), задаёт вопросы | Обсуждают, предлагают варианты |
| Целеполагание (5 мин) | Объявляет тему и цели занятия | Записывают тему, формулируют ожидания |
