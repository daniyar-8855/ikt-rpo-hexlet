Задание к занятию 32. Анализ данных с помощью ИИ

Вариант — по номеру в журнале (свой набор данных на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.
Важно: используй только обезличенные, тестовые данные — без реальных ФИО, телефонов, ID.

Обязательная часть (до 85 баллов)

  1. Возьми набор данных своего варианта (6–8 значений с контекстом).
  2. Составь грамотный запрос к ИИ для анализа (данные + контекст + конкретный вопрос).
  3. Получи ответ ИИ и выпиши его выводы/гипотезы.
  4. Проверь хотя бы один вывод по самим данным: подтверждается или нет.
  5. Сформулируй свой итоговый вывод (2–3 предложения).
Продвинутая часть (+15 баллов): найди в ответе ИИ сомнительное или ошибочное утверждение и докажи числами, что оно неверно. Защити устно.

Наборы данных по вариантам

  • В1. Помесячные продажи интернет-магазина.
  • В2. Посещаемость занятий по неделям.
  • В3. Время отклика сайта по дням.
  • В4. Число заявок в поддержку по месяцам.
  • В5. Средний балл группы по предметам.
  • В6. Количество ошибок в сборках проекта по неделям.

Как сдать: загрузи отчёт (запрос к ИИ, ответ, твоя проверка и вывод), укажи номер варианта. На занятии — устная защита: как проверял вывод ИИ.
