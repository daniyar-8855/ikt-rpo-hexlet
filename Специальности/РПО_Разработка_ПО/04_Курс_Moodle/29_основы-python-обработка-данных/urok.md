# Основы Python для обработки данных

Python — главный язык для работы с данными: от простого подсчёта до анализа и ИИ. Тебе не нужно быть гуру языка, чтобы начать. Достаточно понять несколько строительных блоков — и ты уже можешь читать список чисел, считать суммы и средние, фильтровать и приводить данные в порядок. Этот урок — про базовый Python именно под задачи обработки данных.

## Чему ты научишься

- хранить данные в переменных, списках и словарях;
- проходить по данным циклом и фильтровать их условием;
- считать простые показатели (сумма, среднее, количество);
- понимать, какой блок кода за что отвечает.

## Данные нужно где-то хранить

Самые частые «контейнеры» для данных:

- **переменная** — одно значение: `cena = 1500`;
- **список** (`list`) — упорядоченный набор: `ceny = [1500, 2000, 1800]`;
- **словарь** (`dict`) — пары «ключ — значение»: `tovar = {"name": "Мышь", "cena": 1500}`.

Список — это как столбец таблицы, словарь — как одна строка с подписанными полями. Из них собираются почти любые данные.

## Проход по данным: цикл

Чтобы что-то сделать с каждым элементом, используют **цикл** `for`:

```python
ceny = [1500, 2000, 1800]
itog = 0
for c in ceny:
    itog = itog + c
print(itog)   # 5300
```

Цикл взял каждую цену по очереди и прибавил к сумме. Так считают итоги, количество, ищут максимум.

## Фильтрация: условие

**Условие** `if` оставляет только нужное:

```python
ceny = [1500, 2000, 1800]
dorogie = []
for c in ceny:
    if c >= 1800:
        dorogie.append(c)
print(dorogie)   # [2000, 1800]
```

## Простые показатели

Для списка чисел Python уже умеет:

```python
ceny = [1500, 2000, 1800]
print(sum(ceny))        # 5300 — сумма
print(len(ceny))        # 3 — количество
print(sum(ceny)/len(ceny))  # 1766.6... — среднее
print(max(ceny), min(ceny)) # 2000 1500
```

Это и есть основа обработки данных: посчитать, отфильтровать, свести.

### Мини-кейс
Дан список оценок группы: `[4, 5, 3, 5, 2]`. Нужно найти средний балл и количество отличников (оценка 5). Средний = `sum/len = 19/5 = 3.8`. Отличников — пройти циклом с условием `if o == 5` и посчитать: их 2.

## Частые ошибки

**Ошибка:** делить сумму на `len` без проверки пустого списка.
**Почему:** на пустом списке `len = 0` → деление на ноль, программа падает.
**Как правильно:** сначала проверить `if len(spisok) > 0`.

**Ошибка:** путать список и словарь (брать `tovar[0]` у словаря).
**Почему:** у словаря доступ по ключу, а не по номеру.
**Как правильно:** список — по индексу `[0]`, словарь — по ключу `["name"]`.

## Итог: что теперь делать

- Выбирай контейнер по задаче: список — набор, словарь — запись с полями.
- Проходи данные циклом `for`, отбирай условием `if`.
- Считай итоги через `sum`, `len`, `max`, `min`.
- Проверяй пустые данные, прежде чем делить или брать элемент.
