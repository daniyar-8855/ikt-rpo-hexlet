## Упражнение 2. Прочитай цикл
Что выведет код?
```python
nums = [10, 20, 30]
s = 0
for n in nums:
    s = s + n
print(s)
```

**Критерий приёмки:** 60 (сумма элементов).

## Упражнение 3. Добавь фильтр
Дан список `[3, 8, 1, 9, 4]`. Напиши (словами или кодом), как оставить только числа больше 5.

**Критерий приёмки:** цикл `for` + условие `if n > 5` + `append`; результат `[8, 9]`.

## Упражнение 4. Посчитай показатели
Для списка `ball = [4, 5, 3, 5, 2]` назови: сумму, количество, средний балл, максимум.

**Критерий приёмки:** sum=19; len=5; среднее=3.8; max=5.

## Упражнение 5. Найди ошибку
```python
spisok = []
print(sum(spisok)/len(spisok))
```
Что не так и как исправить?

**Критерий приёмки:** деление на ноль (len=0); добавить проверку `if len(spisok) > 0`.
