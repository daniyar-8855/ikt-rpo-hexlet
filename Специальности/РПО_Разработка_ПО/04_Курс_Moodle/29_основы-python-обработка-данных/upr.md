### Пара 1. Показатели по нескольким числовым наборам

Научись считать базовые показатели и защищать код от пустого списка. Сделай это для **нескольких** наборов, а не одного.

Образец корректного кода (по нему оформляй свои решения):

```python
def pokazateli(chisla):
    """Печатает сумму, количество, среднее, максимум и минимум списка.
    На пустом списке не падает, а сообщает, что данных нет."""
    if len(chisla) == 0:
        print("Список пуст — считать нечего")
        return
    print("Сумма:", sum(chisla))
    print("Количество:", len(chisla))
    print("Среднее:", sum(chisla) / len(chisla))
    print("Максимум:", max(chisla))
    print("Минимум:", min(chisla))

pokazateli([1500, 2000, 1800])   # сумма 5300, среднее 1766.6..., макс 2000, мин 1500
pokazateli([])                    # Список пуст — считать нечего
```

### Пара 2. Фильтрация, выборка по условиям и список словарей

```python
studenty = [
    {"name": "Студент 1", "ball": 78},
    {"name": "Студент 2", "ball": 92},
    {"name": "Студент 3", "ball": 64},
    {"name": "Студент 4", "ball": 88},
    {"name": "Студент 5", "ball": 95},

# сортируем копию списка по полю "ball" по убыванию и берём топ-3
top3 = sorted(studenty, key=lambda s: s["ball"], reverse=True)[:3]

    print(s["name"], "-", s["ball"])
# Студент 5 - 95 / Студент 2 - 92 / Студент 4 - 88
```

**Задание 3.1.** Создай набор из 8–10 записей минимум с двумя числовыми полями (например, `summa` и `tovarov`). Заложи в него хотя бы одну аномалию.

**Задание 3.4.** Выведи короткий **отчёт** с подписями: всего записей, итог по полю, среднее (округли до 1 знака), день/запись с максимумом, список аномалий.

**Задание 3.5.** Добавь защиту от пустого набора и проверь на `[]`. Подготовься объяснить код устно: что делает каждый блок и где (если применял) помогал ИИ.

Образец корректного кода-каркаса (дополни своими условиями аномалий):

```python
prodazhi = [
    {"den": "Пн", "summa": 12000},
    {"den": "Вт", "summa": 9500},
    {"den": "Ср", "summa": 0},
    {"den": "Чт", "summa": 14300},
    {"den": "Пт", "summa": 98000},
    {"den": "Сб", "summa": 16100},
]

if len(prodazhi) == 0:                       # защита от пустого набора
    print("Данных нет")
else:
    summy = [z["summa"] for z in prodazhi]   # вытащили числовое поле
    itogo = sum(summy)
    srednee = itogo / len(summy)

    maks = prodazhi[0]                        # запись с максимумом
    for z in prodazhi:
        if z["summa"] > maks["summa"]:
            maks = z

    anomalii = []                             # аномалии: резко выше среднего или ноль
    for z in prodazhi:
        if z["summa"] > 2 * srednee or z["summa"] == 0:
            anomalii.append(z)

    print("Всего записей:", len(prodazhi))    # 6
    print("Итог:", itogo)                     # 149900
    print("Среднее:", round(srednee, 1))      # 24983.3
    print("Максимум:", maks["den"], maks["summa"])  # Пт 98000
    print("Аномалии:")
    for z in anomalii:
        print("  ", z["den"], z["summa"])     # Ср 0 ; Пт 98000
```

**Критерии приёмки (Пара 3):**
- набор — список словарей из 8–10 обезличенных записей минимум с одним числовым полем и заложенной аномалией;
- показатели (сумма, среднее, максимум, минимум) посчитаны верно;
- аномалии находятся кодом по чётко сформулированному условию, и студент объясняет это условие;
- выведен читаемый отчёт с подписями (среднее округлено);
- есть защита от пустого набора, проверенная на `[]`;
- работа выполнена в Google Colab; студент прошёл устную защиту и указал, где применял ИИ и что проверил сам.
