## 2. Цели, задачи

**Цель:** Сформировать базовые навыки обработки данных на Python: хранить данные в подходящих контейнерах, проходить по ним циклом, отбирать условием и считать простые показатели.

**Задачи:**
- научить выбирать контейнер данных (переменная, список, словарь) под задачу;
- отработать проход по списку циклом `for` и отбор элементов условием `if`;
- освоить функции `sum`, `len`, `max`, `min` для расчёта показателей;
- сформировать привычку проверять пустые данные перед делением и доступом по индексу.

## 3. Ожидаемые результаты

Обучающийся:
- различает переменную, список и словарь и выбирает контейнер по задаче;
- читает и пишет цикл `for` с условием `if` и методом `append`;
- считает сумму, количество, среднее, максимум и минимум списка;
- объясняет и предупреждает типичную ошибку деления на ноль на пустом списке.
