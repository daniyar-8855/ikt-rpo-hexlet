Основы Python для обработки данных

Python — главный язык для работы с данными: от простого подсчёта до анализа и ИИ. Не нужно быть гуру языка, чтобы начать. Достаточно нескольких строительных блоков — и ты уже читаешь список чисел, считаешь суммы и средние, фильтруешь данные.

Чему ты научишься: хранить данные (переменные, списки, словари); проходить циклом и фильтровать; считать сумму, среднее, количество.

Где хранить данные

  • переменная — одно значение: cena = 1500;
  • список (list) — набор: ceny = [1500, 2000, 1800];
  • словарь (dict) — пары ключ-значение: tovar = {"name": "Мышь", "cena": 1500}.

Список — как столбец таблицы, словарь — как одна строка с подписанными полями.

Проход по данным: цикл

ceny = [1500, 2000, 1800]
itog = 0
for c in ceny:
    itog = itog + c
print(itog)   # 5300

Цикл взял каждую цену и прибавил к сумме — так считают итоги, количество, максимум.

Фильтрация: условие

dorogie = []
for c in ceny:
    if c >= 1800:
        dorogie.append(c)
print(dorogie)   # [2000, 1800]

Простые показатели

print(sum(ceny))             # 5300 — сумма
print(len(ceny))             # 3 — количество
print(sum(ceny)/len(ceny))   # 1766.6 — среднее
print(max(ceny), min(ceny))  # 2000 1500
ФункцияЧто делает
sum()сумма чисел
len()количество элементов
max() / min()наибольшее / наименьшее

Мини-кейс

Оценки группы [4, 5, 3, 5, 2]. Средний балл = 19/5 = 3.8. Отличников (оценка 5) — пройти циклом с if o == 5: их 2.

Ошибка: делить сумму на len без проверки пустого списка.
Почему: len = 0 → деление на ноль, программа падает.
Как правильно: сначала проверить if len(spisok) > 0.
Ошибка: путать список и словарь.
Почему: у словаря доступ по ключу, а не по номеру.
Как правильно: список — по индексу [0], словарь — по ключу ["name"].
Итог — что теперь делать:
  • выбирай контейнер: список — набор, словарь — запись с полями;
  • проходи данные циклом for, отбирай условием if;
  • считай итоги через sum, len, max, min;
  • проверяй пустые данные перед делением и доступом.
