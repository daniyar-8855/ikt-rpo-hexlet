Задание к занятию 29. Обработка списка данных на Python

Вариант — по номеру в журнале (свой набор данных на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Возьми набор данных своего варианта (список чисел).
  2. Посчитай: сумму, количество, среднее, максимум и минимум.
  3. Отфильтруй элементы по условию варианта (цикл for + if).
  4. Защити от пустого списка (проверка len > 0 перед делением).
  5. Покажи результат через print и кратко поясни код.
Продвинутая часть (+15 баллов): представь данные как список словарей (поле + значение), посчитай показатель по выбранному полю и выведи топ-3. Защити устно.

Наборы данных и условие фильтра по вариантам

  • В1. Цены товаров — оставить дороже среднего.
  • В2. Оценки группы — посчитать число отличников (== 5).
  • В3. Температуры за неделю — оставить дни выше нуля.
  • В4. Время выполнения задач (мин) — найти задачи дольше 30.
  • В5. Возраст участников — оставить совершеннолетних (≥ 18).
  • В6. Баллы за тест — оставить прошедших порог (≥ 60).
