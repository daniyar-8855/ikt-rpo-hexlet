# Поурочный план. Занятие 34. Автоматизация через API

**РО:** 2.3 · **Связь с ПМ:** Разработка программных модулей · **Объём:** 4 ч (2 пары) · **Сложность:** ★★

## Пара 1 — Что такое API (90 мин)

| Этап | Время | Деятельность |
|---|---|---|
| Орг. момент, цель | 5 мин | Как программы общаются между собой |
| Актуализация | 10 мин | Аналогия «кафе/меню» |
| Теория + демо | 35 мин | API; запрос (URL, метод, параметры); JSON-ответ |
| Практика | 30 мин | Упр. 1–3: разбор запроса, чтение JSON |
| Рефлексия | 10 мин | GET vs POST |

## Пара 2 — Применение и задание (90 мин)

| Этап | Время | Деятельность |
|---|---|---|
| Повторение | 5 мин | Структура запроса |
