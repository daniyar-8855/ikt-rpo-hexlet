Задание к занятию 27. Библиотека промптов для кодовых задач

Вариант — по номеру в журнале (свой набор задач на ~5 человек). № 1–5 → В1 · 6–10 → В2 · 11–15 → В3 · 16–20 → В4 · 21–25 → В5 · 26–30 → В6.

Обязательная часть (до 85 баллов)

  1. Для 3 задач своего варианта составь по структурированному промпту (роль, задача, вход/выход, ограничения, формат, пример).
  2. Покажи «плохой» и «хороший» промпт для одной задачи и разницу в ответах ИИ.
  3. Сделай итерацию: возьми неидеальный ответ и улучши промпт; покажи «до/после».
  4. Оформи мини-библиотеку из 3 готовых промптов с пояснением, когда применять.
Продвинутая часть (+15 баллов): собери шаблон промпта-«заготовки» для типовых кодовых задач (с плейсхолдерами) и продемонстрируй на 2 примерах. Защити устно.

Наборы задач по вариантам

  • В1. Валидация e-mail; реверс строки; факториал.
  • В2. Проверка палиндрома; подсчёт гласных; min/max.
  • В3. Парсинг «ключ=значение»; сумма списка; уникальные элементы.
  • В4. Сортировка по убыванию; перевод C↔F; счётчик слов.
  • В5. Безопасное деление; проверка чётности; срез списка.
  • В6. Генерация пароля; проверка длины строки; объединение списков.

Как сдать: загрузи отчёт с промптами и ответами ИИ, укажи номер варианта. На занятии — короткая устная защита.

СРС (самостоятельная работа, 2 ч)

Тема СРС: Библиотека промптов для кодовых задач.
Задание: Собери 5–7 эффективных промптов для типовых кодовых задач с пояснением. Используй обезличенные данные; укажи, если применял ИИ.
Критерий приёмки: промпты содержат роль, задачу, контекст и формат, есть пояснение чем хорош каждый.
