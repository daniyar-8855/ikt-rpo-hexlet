# План учебного занятия

**Организация:** ТОО «Колледж Хекслет Казахстан»  
**Дисциплина:** Применение информационно-коммуникационных и цифровых технологий  
**Тема занятия:** Защита итогового проекта и рефлексия  
**Подготовил педагог:** ____________________  
**Дата:** «___» __________ 20__ г.

## 1. Общие сведения

- **Курс, группа:** ____________________ (квалификация 4S06130103 «Разработчик ПО»)
- **Тип занятия:** урок контроля / защита проекта
- **Продолжительность:** 2 часа (90 мин)
- **Результат обучения:** РО 2.3 · **Связь с ПМ:** Разработка программных модулей

## 2. Цели, задачи

**Цель:** Научить защищать итоговый проект по понятной структуре, честно показывать роль ИИ и проводить рефлексию своей работы, показать связь с будущей профессией.

**Задачи:**
- сформировать умение строить устную защиту по схеме «задача → решение → инструменты ИИ → результат → выводы»;
- научить честно указывать роль ИИ и отвечать на вопросы по своей работе;
- развить навык рефлексии (что получилось, что трудно, что улучшить, чему научился);
- показать связь с профессией (защита решений перед командой/заказчиком, ретроспектива).

## 3. Ожидаемые результаты

Обучающийся:
- строит устную защиту проекта по заданной структуре;
- честно объясняет, где помогал ИИ и что проверил сам;
- спокойно отвечает на вопросы по своей работе;
- проводит рефлексию по четырём вопросам.

## 4. Необходимые ресурсы

- Курс в Moodle: страница «Теория», тест и задание занятия 39.
- Схемы: «Структура устной защиты», «Цикл рефлексии» (папка assets).
- Компьютер, проектор, доступ к Moodle и интернету.

## 5. Ход занятия

| Этап / Время | Действия педагога | Действия обучающегося |
|---|---|---|
| Организационный момент (5 мин) | Приветствие, проверка готовности и присутствующих | Занимают места, готовят свои проекты |
| Мотивация (10 мин) | Вводит практической ситуацией («сделал проект — теперь нужно защитить»), задаёт вопросы | Обсуждают, делятся опасениями |
| Целеполагание (5 мин) | Объявляет тему, цели и порядок защиты | Записывают тему, формулируют ожидания |
| Изучение нового материала (35 мин) | Разбирает структуру защиты и цикл рефлексии по схемам, показывает мини-кейс, акцент на честном указании роли ИИ | Слушают, работают со схемами, готовят план выступления |
| Закрепление (15 мин) | Организует мини-защиты в парах и разбор типичной ошибки | Репетируют защиту друг другу, дают обратную связь |

| Задание (5 мин) | Поясняет порядок сдачи защиты и письменной рефлексии, критерии | Записывают, уточняют |
