# План занятия 20. Профессиональные ресурсы разработчика

> Для преподавателя (в Moodle — скрытая «Страница»). Тип: комбинированное, 2 ч. РО 2.2.

- **Дисциплина:** Применение ИКТ и цифровых технологий (РПО, 4S06130103)
- **Тема:** Профессиональные ресурсы разработчика (Stack Overflow, MDN, GitHub)
- **Цель:** научить пользоваться ключевыми ресурсами и оценивать найденное.
- **Результат обучения:** РО 2.2.

## Ожидаемые результаты
Студент умеет: выбирать ресурс под задачу; оценивать ответ на SO; ориентироваться в GitHub; не копировать код без понимания.

## Ход занятия (90 мин)
| Этап | Время | Содержание |
|---|---|---|
| Орг. момент | 5 мин | Зацепка: «разработчик не помнит всё, а умеет найти» |
| Теория | 20 мин | Документация, Stack Overflow, GitHub |
| Демонстрация | 15 мин | Поиск решения в документации и SO; обзор GitHub |
| Практика | 30 мин | Упражнения 1–5 (выбор источника, оценка ответа, маршрут поиска) |
| Выдача задания | 10 мин | Решение задачи через ресурсы по вариантам |
| Итог, рефлексия | 5 мин | Что делать / где применяется / как не ошибиться |

## Средства обучения
Проектор, ПК с доступом к MDN, Stack Overflow, GitHub.

## Контроль
- Текущий: упражнения, тест (банк 12, 10 случайных).
- Задание: решение задачи через ресурсы по вариантам + устная защита.

## Источники
- ГОСО ТиПО (приказ МП РК № 348); MDN; Stack Overflow; GitHub Docs.

## Адаптация (при необходимости)

При наличии обучающихся с особыми образовательными потребностями предусматриваются адаптация темпа, дополнительные пояснения и индивидуальные задания, одобренные методическим объединением.
